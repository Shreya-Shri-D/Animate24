# Animate24

Short 2D animation project built in **Synfig Studio**, created for ANIMATE 2024 online 2D animation hackathon organized by FOSSEE, IIT Bombay. The piece uses **UN Sustainable Development Goal 13 (Climate Action)** as its theme, following a simple narrative about rising water, place, and people. This project was recognised as Good Submission under Climate Action - AI Images.

## What’s in this repo

| Type | Description |
|------|-------------|
| `*.sifz` | Synfig scene files (compressed). Open in [Synfig Studio](https://www.synfig.org/) to edit or render. |
| `*.png` / `*.jpg` | Artwork and backgrounds: characters, buildings, water, sky, and scene plates. |

Notable scene files include intro, dialogue, rising water, flood beat, and ending sequences—use them as a map of the story’s structure.

## Requirements

- [Synfig Studio](https://www.synfig.org/) (1.4.x or newer recommended) to open `.sifz` files and export video.

## Quick start

1. Clone this repository.
2. Install Synfig Studio.
3. Open any `.sifz` file from this folder (e.g. `final.sifz` or `animate-2.sifz`).
4. Preview in Synfig, then render to your preferred format (e.g. PNG sequence or MP4 via toolchain of your choice).
